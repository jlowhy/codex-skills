# Style References

Use the notes vault as the source of truth for Jonathan's writing-style references.

## Source Set

Load:

- `~/Notes/Notes/Writing References - Jonathan Voice.md`

Use only Jonathan's past articles, posts, drafts, and channel notes linked from `Writing References - Jonathan Voice.md`.

## How To Use

- Open `Writing References - Jonathan Voice.md` and use it as the voice anchor for all style work. Open 2-4 linked samples that best match the current piece, prioritizing published samples over drafts.
- Treat venue as an adaptation layer after the voice anchor is loaded: LinkedIn, X, Bookface, Hacker News, Reddit, articles, and drafts differ in length, density, and call-to-action, but Jonathan's voice remains primary.
- If the user asks for wordsmithing, tone options, style calibration, voice matching, or examples of possible directions, derive the options from Jonathan's samples and the current piece's venue.
- If the user asks for structure, argument, or diagnosis only, do not load style references unless a style mismatch is part of the problem.
- Use the virtual panel as editorial hats for diagnosis and taste disagreement, not as a source-reading instruction.
- Do not browse for or load outside style references.
- For substantial rewrites, generate 3-5 distinct directions, then tighten only after the user chooses.
- Do not copy surface markers from any one Jonathan sample if they do not fit the current piece.
